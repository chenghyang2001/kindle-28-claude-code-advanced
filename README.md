## 使用說明
1. 從 `demo/[01-快速上手]/` 開始
2. 完成練習後將解答存入該課 `answer/`
3. 執行 `/study-scaffold carry N` 把答案攜帶到下一課 `starter/`
4. 填寫每課 `STEP_LOG.md` 記錄學習過程
5. 完成所有課程後，`docs/踩坑紀錄.md` 會是最寶貴的個人筆記
