# 練習 01 — 基礎：啟用一個官方外掛並實跑

## 情境說明
官方外掛把常見工作流（審查、簡化、feature dev）封裝成可直接呼叫的能力。

## 範例：外掛操作
```text
/plugin            # 開啟外掛管理（瀏覽 marketplace / 安裝 / 啟用）
/code-review       # 對目前 diff 跑審查
/simplify          # 對改動的程式碼做簡化（只動品質不找 bug）
```

## 任務
### 任務 1
啟用一個官方外掛（如 code review），說明它的用途與適用時機。

### 任務 2
對一段真實程式碼跑一次，整理它的輸出（必改 / 建議 / 備註分類）。

## 驗收標準
- [ ] 成功啟用並實跑一個外掛
- [ ] 輸出有經過你的整理分類（非原樣貼上）
