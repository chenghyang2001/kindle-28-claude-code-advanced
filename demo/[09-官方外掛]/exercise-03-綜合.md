# 練習 03 — 綜合挑戰：外掛組合決策表

> 選配：完成前兩個練習後再挑戰

## 挑戰情境
為你的工作流挑 3 個外掛，寫一張「何時用哪個」的決策表。

## 範例：決策表格式
```markdown
| 外掛 | 觸發條件（何時用） | 不該用的反例 |
|------|------------------|-------------|
| code-review | PR 前、改動 >= 2 檔 | 改一行 typo |
| simplify    | 程式已正確、想去冗餘 | 還有未修 bug 時 |
| feature-dev | 從零做新功能 | 既有 bug 修復 |
```

## 限制條件
- 決策表需有「觸發條件」欄
- 需標明每個外掛「不該用」的反例

## 完成後
將解答存入 `answer/ex03-answer.md`。
