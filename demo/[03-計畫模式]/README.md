## 課程結構
- `exercise-01-基礎.md`：對一個中等任務（如『加一個 CLI 子指令』）先請 Claude 產出計畫，再決定是否執行。
- `exercise-02-進階.md`：為一個專案建立 .claude/ 的 pre-setup（如慣用指令、目錄說明），讓後續任務更易一次到位。
- `exercise-03-綜合.md`：把一個你過去『做歪』的任務，重做成『先計畫 + 審查 + 執行』三段式，記錄差異。
