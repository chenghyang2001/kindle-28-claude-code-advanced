# 練習 03 — 綜合挑戰：跨對話的脈絡接續機制

> 選配：完成前兩個練習後再挑戰

## 挑戰情境
為一個跨多次對話的任務，建立「脈絡接續」機制：把關鍵狀態落檔，讓新 session 能 2-3 句還原進度。

## 範例：落檔結構
```text
memory/
├── MEMORY.md            # 長期索引（一行一則）
└── short-term/
    └── 2026-06-20.md    # 當日快照：進度 + 關鍵 ID + 待辦
```

## 限制條件
- 機制需可跨 session 還原進度（新 session 讀檔即接上）
- 需說明哪些資訊落檔、哪些可丟棄
