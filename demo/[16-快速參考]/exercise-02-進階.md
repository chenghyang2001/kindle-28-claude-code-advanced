# 練習 02 — 進階：補上 flags 與分類

## 情境說明
小抄加上 CLI flags 與分類，才能隨手查、查得快。

## 範例：CLI flags
```bash
claude -p "..."                 # 非互動、印完即走
claude --model <id>             # 指定模型
claude --resume                 # 接續 session
claude mcp list                 # 列出 MCP server
```

## 任務
### 任務 1
為小抄每個指令補上常用 flag。

### 任務 2
把小抄分類整理（檔案操作 / session 控制 / 自動化 / 外掛）。

## 延伸思考
哪些指令值得設成 alias？哪些靠記憶反而更快（設 alias 是過度工程）？
