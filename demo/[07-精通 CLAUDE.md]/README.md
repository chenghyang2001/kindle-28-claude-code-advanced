## 課程結構
- `exercise-01-基礎.md`：為一個現有專案寫一份精煉的 CLAUDE.md（專案概述 + 架構 + 執行方式 + 關鍵決策）。
- `exercise-02-進階.md`：把臃腫的 CLAUDE.md 模組化：用 @import 拆成多個 instructions 檔。
- `exercise-03-綜合.md`：為一個團隊設計 CLAUDE.md 規範：哪些全域、哪些專案、哪些個人，如何避免衝突。
