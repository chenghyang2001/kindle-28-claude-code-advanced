# 練習 03 — 綜合挑戰：端到端自動化規格

> 選配：完成前兩個練習後再挑戰

## 挑戰情境
為一個真實重複任務，組合「skill + subagent + 排程」做出端到端自動化規格。

## 範例：排程觸發（cron 概念）
```text
# 每天 06:00 跑一次（無人值守）
0 6 * * *  cd /path/to/repo && claude -p "/daily-report" >> log 2>&1
護欄：逾時上限、失敗通知、單例鎖（防重入）
```

## 限制條件
- 需標明各環節的失敗處理（某步掛了怎麼辦）
- 排程任務需考量無人值守的護欄（逾時 / 重試 / 通知）
