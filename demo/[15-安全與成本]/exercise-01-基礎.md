# 練習 01 — 基礎：依任務選對 model 控成本

## 情境說明
不同任務該配不同 model：簡單格式化用便宜的，複雜推理才上貴的。

## 範例：成本量級對照（相對）
```text
任務            建議 model        理由
格式化/摘要/通知  Haiku            最便宜，夠用
分析/程式碼生成   Sonnet           平衡
複雜推理/架構     Opus             最強但最貴，謹慎用
指定方式：claude --model claude-haiku-4-5-20251001
```

## 任務
### 任務 1
為三種任務（格式化 / 分析 / 複雜推理）各選一個最划算的 model 並說明理由。

### 任務 2
估算三種選擇的相對成本差距（量級即可）。

## 驗收標準
- [ ] 三任務各有 model 選擇 + 理由
- [ ] 有相對成本的量級比較
