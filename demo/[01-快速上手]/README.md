# [01] 快速上手（Quick Start）

> 所屬：Part I — 基礎　|　難度：基礎　|　預估：~30 分鐘

## 學習目標
- 掌握 5 個核心提示技巧
- 做出第一個一次到位（one-shot）的工作流
- 養成提供脈絡而非逐步指令的習慣

## 前置條件
- 無

## 課程結構
- `exercise-01-基礎.md`：你要請 Claude 把一個資料夾內所有 .txt 檔合併成一個檔。先用最隨意的一句話試一次，再用結構化方式試一次。
- `exercise-02-進階.md`：設計一個能『一次到位』完成「讀 CSV → 算每欄平均 → 輸出 Markdown 表」的 prompt，不靠來回追問。
- `exercise-03-綜合.md`：把你最常重複的一個任務，寫成一個可重用的 prompt 範本（含參數占位符）。

## 完成標準
- [ ] 閱讀本課 README
- [ ] 完成基礎練習，解答存入 `answer/ex01-answer.md`
- [ ] 完成進階練習，解答存入 `answer/ex02-answer.md`
- [ ] （選配）完成綜合挑戰，解答存入 `answer/ex03-answer.md`
- [ ] 填寫 `STEP_LOG.md`
- [ ] 執行 `/study-scaffold carry 1` 攜帶答案到下一課

## 本課重點概念
- one-shot 執行
- 明確的成功標準
- 給脈絡不給流程
- 產出後自我驗證
- 反模式：模糊指令
