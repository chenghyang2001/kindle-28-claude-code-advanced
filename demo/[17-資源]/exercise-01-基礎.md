# 練習 01 — 基礎：建立持續學習來源清單

## 情境說明
工具演進快，固定幾個高品質來源持續追蹤，比四處亂看有效。

## 範例：來源類型
```text
官方文件 / changelog   ：權威、最即時的功能變更
GitHub repo / issues   ：實作細節、社群踩坑
社群（Discord 等）     ：第一手技巧與討論
精選電子報 / 影片       ：經過篩選的重點
```

## 任務
### 任務 1
整理 5 個你會持續追蹤的 Claude Code 學習來源與追蹤理由。

### 任務 2
標出更新最頻繁、最值得設提醒的那一個。

## 驗收標準
- [ ] 5 個來源各有追蹤理由
- [ ] 標出最高優先來源

## 完成後
將解答存入 `answer/ex01-answer.md`。
