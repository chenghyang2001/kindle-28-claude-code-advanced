# 練習 03 — 綜合挑戰：用 GSD 跑一個真實小專案

> 選配：完成前兩個練習後再挑戰

## 挑戰情境
挑一個真實小專案，完整走一遍 GSD：每階段產出交付物，最後產一份整合 manifest。

## 範例：整合 manifest 骨架
```markdown
# MANIFEST — <專案名>
## 各階段交付物
- Phase 1：<連結/路徑>
- Phase 2：<連結/路徑>
## 已知限制 / 未解問題
## 下一步
```

## 限制條件
- 每階段須有可驗收的交付物（非口頭「做完了」）
- 最終須有整合 manifest 串起所有階段
