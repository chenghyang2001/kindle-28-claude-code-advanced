# [05] GSD 框架（GSD (Get Shit Done)）

> 所屬：Part III — 框架　|　難度：進階　|　預估：~50 分鐘

## 學習目標
- 理解 GSD 框架的階段切分
- 用 phases 拆解長型專案
- 在 GSD 中協調 multi-agent

## 前置條件
- 完成 [02] 與 [03]。

## 課程結構
- `exercise-01-基礎.md`：把一個會跑很久的專案（如『重構一個模組』）拆成 GSD 的明確階段。
- `exercise-02-進階.md`：在其中一個階段引入 multi-agent 分工，指定各 agent 的邊界與交付物。
- `exercise-03-綜合.md`：用 GSD 跑一個真實小專案，產出每階段的交付物與最終整合報告。
