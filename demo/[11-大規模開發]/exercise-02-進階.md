# 練習 02 — 進階：規劃一次安全重構

## 情境說明
安全重構＝不改外部行為、只改內部結構，並用測試當安全網逐步推進。

## 範例：重構步驟
```text
1. 先補/確認 characterization test（釘住現有行為）
2. 小步改：每次一個關注點，跑測試保持綠
3. git commit 每個綠燈檢查點（可隨時回滾）
4. 完成後對照行為：對外輸出不變
```

## 任務
### 任務 1

### 任務 2
說明你如何用測試當安全網（哪些測試保證行為不變）。
